## Tools I have access to for work

Amazon Q
GitHub Copilot
Claude code
Cursor

---

### Amazon Q

Pros:

- Access to many models
- Insight into AWS tooling

Cons:

- My work isn't heavily involved on the AWS side of things
- It's implemented through a VSCode extension
- Not very good UX

### GitHub Copilot

Pros:

- Ghost text/tab complete is decent enough
- Not intrusive, I stay in control
- Limited access to using models/agents
- Not locked into using a particular IDE

Cons:

- Ghost text/tab autocomplete isn't the best
- $10/month

### Claude code

Pros:

- Best coding tool available for agentic flows

Cons:

- No ghost text/tab complete
- UI isn't the greatest

### Cursor

Pros:

- All-in-one package: (ghost text/tab complete + agentic abilities)
- Can choose between multiple models
- Best ghost text/tab complete available

Cons:

- Locked into using VSCode
